BOSCO

Bosco is a utility knife to help manage the complexity that using microservices, which naturally results in a large number of code repositories, brings with it.

Bosco is inspired by the Github 'setup', e.g. can a developer run one simple command and get up and running?

Features

  • Configurations for Github organizations and teams
  • Aggregation of all static assets to be served locally emulating a CDN functionality
  • Push assets to S3 to be used with Compoxure
  • Detection of duplicate libraries when using the CDN feature
  • Support for local commands to handle your repos
